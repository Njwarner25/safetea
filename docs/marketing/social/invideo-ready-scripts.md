# SafeTea — InVideo-Ready Scripts (5)

**Tool:** InVideo AI / InVideo Studio
**Format:** 9:16 vertical, 15–35s
**Voice model recommendation:** InVideo AI voice "Aria" or "Olivia" — calm, mid-pitch, US English. Pace: 1.0× (do not speed up; the brand voice is calm).
**Music recommendation:** InVideo library → search "minimal documentary," "subtle suspense," or "cinematic ambient." Volume: -18 dB under voiceover.
**Caption styling:** Bold sans-serif (Inter, Montserrat), white text, black drop-shadow or 80% black box behind. ALL on-screen text in the script below should be added as text overlays — InVideo auto-syncs these to your script if you paste it in script mode.
**End card on every video:** 2-second hold of `safetea.app — link in bio` over a soft fade-in of the SafeTea logo. Save as a reusable end-card in InVideo Brand Kit.

How to use this file with InVideo:
1. Open InVideo AI or InVideo Studio.
2. For AI mode: paste the **Voiceover** column straight into the prompt; manually replace its scene picks using the **Stock-footage search terms** below.
3. For Studio mode: build scene-by-scene using the shot list. Each row in the table = one scene.
4. Set total duration to match the script. Render at 1080×1920, 30fps.
5. Cross-post the same render to TikTok and Instagram Reels.

## Production tips for InVideo specifically

- **Voice consistency:** Lock to one AI voice (e.g., "Aria") and use it across all 5 videos. Switching voices between posts confuses the brand identity in followers' feeds.
- **Brand kit upload:** Upload the SafeTea logo + a 2-second end-card animation to InVideo's Brand Kit so every render automatically gets the same outro. Saves ~5 min per video.
- **Color overlay:** Apply a subtle teal overlay (matching SafeTea's brand color) at ~10% opacity on stock clips to unify the visual feel across videos.
- **Captions:** InVideo auto-generates captions from voiceover. Always do a manual proof — AI captions miss "SafeTea" → often transcribed as "Safety" or "Safe Tea." Critical to fix before export.
- **Render settings:** 1080×1920, 30 fps, MP4. Cap each video under 50 MB so TikTok/IG don't recompress aggressively.
- **Posting cadence:** Stagger renders. Post one of these per day for 5 days, not all five at once. Algorithms reward consistent posting more than burst posting.
