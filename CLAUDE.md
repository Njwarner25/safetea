# Claude Code Guidelines

## File Preservation

- **Never overwrite or replace existing files entirely.** Always use targeted edits to modify only the specific lines or sections that need to change.
- Use the `Edit` tool instead of the `Write` tool when modifying existing files.
- Read a file before making changes to understand its full context.
- Do not delete files unless explicitly asked to do so.
- Do not remove existing code, comments, or configuration that is unrelated to the current task.
- When adding new functionality, integrate it into existing files rather than replacing them with new versions.
